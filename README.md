# react-map-component

## Build/Run

#### Requirements

- Node.js
- NPM

```javascript

/* First, Install the needed packages */
npm install

/* Then start the React (Vite) app */
npm run dev

```

#### use mock rest json server using [JSON Server](https://www.npmjs.com/package/json-server)
```javascript

/*  to start the json server */
npm run server

```
